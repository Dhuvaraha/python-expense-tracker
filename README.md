# Expense Tracker CLI

A command-line expense tracking application built using Python with JSON-based data persistence.

## Features

* Add Expense
* View Expenses
* Delete Expense
* View Total Expenses
* Category-wise Expense Summary
* Persistent Storage using JSON

## Technologies Used

* Python
* JSON
* File Handling
* Exception Handling
