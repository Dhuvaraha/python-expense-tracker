# Expense Tracker CLI

A command-line expense tracking application built using Python that helps users manage and track their daily expenses. The application stores data in a JSON file, allowing expenses to persist between sessions.

## Features

- Add new expenses
- View all recorded expenses
- Delete expenses
- Calculate total expenses
- View category-wise expense summaries
- Persistent storage using JSON

## Technologies Used

- Python
- JSON
- File Handling
- Exception Handling

## How to Run

1. Clone the repository

```bash
git clone https://github.com/Dhuvaraha/python-expense-tracker.git
```

2. Navigate to the project folder

```bash
cd python-expense-tracker
```

3. Run the application

```bash
python main.py
```
